# Tunebox

> 飞牛 NAS（fnOS）多平台聚合音乐搜索下载工具。基于开源项目 [guohuiyuan/go-music-dl](https://github.com/guohuiyuan/go-music-dl) 二次开发。

## 功能特性

- **多平台聚合搜索**：网易云、QQ、酷狗、酷我、咪咕、5sing、千千、汽水、JOOX、Jamendo、Bilibili
- **单曲 / 歌单 / 专辑** 三种搜索模式，支持直接粘贴分享链接
- **Web 试听**：内嵌播放器，支持换源、连续播放
- **无损下载**：FLAC、320kbps 等多音质，自动下载歌词与封面
- **本地歌单**：导入在线歌单到本地，便于管理
- **每日推荐**：聚合各平台推荐入口

## 系统要求

- fnOS ≥ 1.1.8
- 架构：x86_64
