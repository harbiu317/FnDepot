# 图床 PicHub

> 飞牛 NAS（fnOS）自托管图床，支持多云存储、相册管理、API Token。

## 功能特性

### 多云存储
- **通用 S3** — 兼容任意 S3 协议的对象存储
- **七牛云 Kodo** / **阿里云 OSS** / **腾讯云 COS** — 国内主流云厂商直连
- 存储设置切换时自动回显已保存配置

### 图片处理
- 自动压缩、WebP 转换、EXIF 清理
- 中文水印（支持 TTF/TTC 字体）
- GIF 元数据剥离

### 相册 & 管理
- 相册分组管理，批量移动 / 删除
- 侧边栏实时显示图片 / 相册数量
- 图库网格预览，支持搜索和筛选

### 分享 & API
- 多格式分享链接：URL / Markdown / HTML / BBCode
- API Token 管理，支持外链密钥上传
- 配额管理：按用户限制空间占用

### 其他
- 首次启动向导：设置管理员账号、最大文件大小
- 深色 / 浅色主题跟随系统

## 系统要求

- fnOS ≥ 1.1.8
- 架构：x86_64
