# 图床 PicHub

自托管图床，支持多云存储、相册、API Token、水印、中文。

## 功能

- **多云存储**：通用 S3、七牛云 Kodo、阿里云 OSS、腾讯云 COS
- **图片处理**：自动压缩、WebP 转换、EXIF 清理、中文水印（TTF/TTC）、GIF 元数据剥离
- **相册 / API Token**：分组管理，外链密钥上传
- **分享链接**：URL / Markdown / HTML / BBCode
- **配额管理**：按用户限制空间占用
- **首次启动向导**：设置管理员账号、最大文件大小

## 访问

安装完成后，首次访问进入向导设置。之后通过桌面图标进入 Web 界面。
